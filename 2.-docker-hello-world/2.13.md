# 2.13 컨테이너 삭제하기

이미 생성된 컨테이너를 삭제하도록 하겠습니다.

| grouq:~ giljae$ docker rm ubuntu ubuntu |
| :--- |


사용법:

| docker rm \[OPTIONS\] CONTAINER \[CONTAINER...\] |
| :--- |


컨테이너 이름 대신에 컨테이너 ID를 사용해도 됩니다.
